# HyperMotion Board

HyperMotion is a custom-designed controller PCB built for robotics applications such as line follower and mini sumo robots.  
It combines motor driving, sensor interfacing, and control logic into a compact, reliable board optimized for competitions.

You can find the Gerber File and EasyEDA Project file on: [https://oshwlab.com/bicicleta11/hypercontrol](https://oshwlab.com/bicicleta11/hypercontrol)

## Features

- **CPU:** ATmega328P-AU (Arduino-compatible)  
- **Motor Driver:** Dual-channel IFX9201SG  
- **Power Input:** Supports **2S–4S LiPo batteries**  
- **USB-C:** For serial communication and firmware upload  
- **DIP Switch:** 3 switches (8 configuration combinations)  
- **Sensor Ports:** 8 input pins with **5V + GND** headers  
- **LEDs:** 2 fully programmable status LEDs  
- **Start/Stop Compatibility:** Supports official start-stop modules for mini sumo robots  

---

## Advantages

- All-in-one design: MCU + motor drivers + sensor inputs  
- Compact and lightweight, reducing wiring complexity  
- Plug-and-play compatibility with robotics modules  
- Reliable performance for competition environments  

---

## Applications

- Line Follower Robots  
- Mini Sumo Robots  
- Prototyping robotics projects requiring motion + sensor integration
